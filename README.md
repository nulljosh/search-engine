# Build Your Own Search Engine

A text search engine with indexing, ranking, and a query interface.

## Scope
- Document ingestion (text/HTML)
- Inverted index construction
- TF-IDF scoring
- Boolean and ranked queries
- Simple web crawler (stretch goal)
- Web UI for search (stretch goal)

## Learning Goals
- Inverted index data structure
- Text tokenization and normalization
- TF-IDF and BM25 ranking algorithms
- Persistent storage of index data
- Information retrieval fundamentals
